Animation basics


Animation
So to create an animation like this we will have to give it a name like i gave "ani1". Then you ll have to use key frames by typing @keyframes on the css page and the name of the animation. Then open two selectors "from" and "to" and then you can set the values like i made it start from 150px too 450px to create this kind of animation. You can set the timing using the animation duration property and you can also set, how many times the animation will run by using animation iteration count.
Now let't talk about the other funcions of this animation
Properties
So the other properties that have been used in this animations are:-
1. Animation fill mode: With this function you can select if the animation is going stop at the end or just continue going. I have selected the forwards option so it stops when the animation is complete.(You'll have to set the animation count to one to apply this properly).
2. Animation timing function: The animation timimg function is used to tell the compiler, when to slow down the animation and when to speed up. There are a lot of options i have selected ease out which means the animation will start slow and will speed up at the end moment.
3.Animation delay: The animation delay is nothing but telling the compiler to start the animation a bit later after loading. I have set the timing for 3 seconds and as a result the animations starts a bit later instead of starting right after loading the page.
4. Animation direction: with this property you can set, like from where should the animation start. From the starting to ending or from the ending to the start.
Syntex
Here we have used the other syntex property which indicates what to do when the animation is that much completed. We have used the % type.
Firts of all i have seleted 0%, means when the animation will start what will happen. then selected 25% and 75% and 100%. So when the animation starts the box starts to move towards right, when the animation is 25% finished you the box will be at complete right. After that the box will move downwards and then on the left side and this will continue.

You can write down all the functions in a single line, the order is like: Animation name, animation duration, animation timing function, animation delay, animation iteration count, animation fill mode;......
